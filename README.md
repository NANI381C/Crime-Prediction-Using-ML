# 🔍 Crime Prediction Using Machine Learning

An end-to-end machine learning pipeline designed to analyze historical crime data, uncover underlying patterns, and predict future crime trends and patterns based on spatial and temporal factors.

---

## 🚀 Key Features

* **Data Preprocessing & Cleaning:** Handling missing data, encoding categorical variables, and structuring temporal features (hour, day, month, year).
* **Exploratory Data Analysis (EDA):** In-depth data analysis with visualizations highlighting crime hotspots, peak time frames, and high-risk categories.
* **Feature Engineering:** Extracting geographical insights and time-series variables to optimize predictive accuracy.
* **Predictive Modeling:** Implementation and evaluation of multiple algorithms, including **Random Forest**, **XGBoost**, and **LSTM** models.
* **Performance Evaluation:** Comparative analysis of models using standard metric scores (Accuracy, Precision, Recall, and F1-Score).

---

## 🛠️ Tech Stack & Libraries

* **Core Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Machine Learning:** Scikit-Learn, XGBoost
* **Deep Learning:** TensorFlow / Keras (LSTM)
* **Visualization:** Matplotlib, Seaborn

---

## 📊 Workflow Architecture

1. **Data Acquisition:** Gathering historical logs and spatial-temporal crime records.
2. **Data Cleaning:** Refining geographic coordinates, dates, and eliminating anomalies.
3. **Feature Selection:** Identifying the strongest correlations between time, location, and crime type.
4. **Model Training:** Training ensemble methods and sequential models on training subsets.
5. **Evaluation:** Benchmarking model performance to determine the most reliable predictive system.
